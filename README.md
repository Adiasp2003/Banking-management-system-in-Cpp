
## Banking Management System in C++

This project is a console-based Banking Management System implemented in C++ using object-oriented programming concepts. It simulates basic banking operations including account creation, user authentication, balance inquiry, deposits, and withdrawals using persistent file-based storage.

### Features
- Account creation and secure login
- Deposit and withdrawal operations
- Balance inquiry
- File-based data persistence
- Modular and object-oriented design

### Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

### Purpose
This project was developed to strengthen C++ programming fundamentals, object-oriented design, and problem-solving skills relevant to entry-level software engineering roles.
